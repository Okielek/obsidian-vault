### `mean()`
returns the average of a vector.
### `length()`
returns the length of a vector (if summable).
### `rowSums()`
Used to calculate the sum of each row in a matrix.
**Usage Example:**
```r
my_matrix <- matrix(1:9, nrows = 3)
sum_matrix <- rowSums(my_matrix)
```
## `sd()`

## `abs()`
