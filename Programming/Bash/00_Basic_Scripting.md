
## Info
Bash always exists in some directory!
## Basic commands
* `echo` — prints out the content back to the terminal
	 * `echo <input>`
* `cd` — change directory
	 * `cd <directory>`
* `pwd` — print working directory
* `ls` — list all files
	 * see: [[01_Detailed_Info#`ls`|details]]
* `touch`  — creates a file
* `rm` — removes file
	 * usg ex: `rm <file>`
	 * see: [[01_Detailed_Info#`rm`|details]]
* `clear` — clears the terminal (CTRL+L)
* `mkdir` — makes a directory
	 * usg ex: `mkdir <directory>`
* `history` — prints terminal history
## File manipulation
* `mv` — moves the file from *file1* to *file2*.  It **overwrites** a file! Can be used to **rename** file.
	 * usg ex:`mv <file1> <file2>` 
## File searching
* `cat` — will print the contents of a *file*.
* `grep` — searches a file/input for a specified pattern and prints it out.
	 * usg ex: `grep <pattern> <file>` 
	 * see:  [[01_Detailed_Info#`grep`|details]]
