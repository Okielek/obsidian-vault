
## Info
Bash always exists in some directory!
## Basic commands
* `echo` тАФ prints out the content back to the terminal
	 * `echo <input>`
	 * see:  [[Detailed_Info#`echo`|details]]
* `cd` тАФ change directory
	 * `cd <directory>`
* `pwd` тАФ print working directory
* `ls` тАФ list all files
	 * see: [[Detailed_Info#`ls`|details]]
* `touch`  тАФ creates a file
* `rm` тАФ removes file
	 * usg ex: `rm <file>`
	 * see: [[Detailed_Info#`rm`|details]]
* `clear` тАФ clears the terminal (CTRL+L)
* `mkdir` тАФ makes a directory
	 * usg ex: `mkdir <directory>`
* `history` тАФ prints terminal history
* `type` тАФ  gives you a type of a command (can be built in or not)
## File manipulation
* `mv` тАФ moves the file from *file1* to *file2*.  It **overwrites** a file! Can be used to **rename** file.
	 * usg ex:`mv <file1> <file2>` 
## File searching
* `cat` тАФ will print the contents of a *file*.
* `less` тАФ prints out the content of a *file* but paginates data.
* `grep` тАФ searches a file/input for a specified pattern and prints it out.
	 * usg ex: `grep <pattern> <file>` 
	 * see:  [[Detailed_Info#`grep`|details]]
* `which` тАФ prints out the firectory of a given file.
